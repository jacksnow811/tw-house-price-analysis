# 第二部分：資料分析
更新時間：2026-05-29　狀態：🚧 尚未開始

> 本專案分兩部分：第一部分＝資料抓取（見 `01_資料抓取.md`）、**第二部分＝資料分析（本檔）**。

---

## 目標
針對第一部分（爬蟲）抓回、存放在 `data/` 的 CSV，做後續的整理與分析。

## 資料來源
- 位置：`data/` 資料夾
- 命名：`{縣市}_{鄉鎮市區}_{社區名稱}_{起始年月}-{結束年月}.csv`
- 編碼：UTF-8-BOM

## 欄位說明（每筆 28 欄）
最前面 3 欄為查詢條件，方便多檔合併時辨識：
| 欄位 | 說明 |
|------|------|
| city / district / keyword | 查詢條件（縣市 / 鄉鎮市區 / 社區名稱）|

基本資料（來自 list.jsp 的 #price_table）：
地段位置或門牌、社區簡稱、總價(萬元)、交易日期、單價(萬元/坪)、總面積(坪)、
主建物佔比(%)、型態、屋齡、樓別/樓高、主要用途、交易標的、建物格局、
車位總價(萬元)、管理組織、電梯、交易/歷次轉移、備註

明細資料（來自每筆「明細」按鈕的 /QueryPrice/detail/ API）：
| 欄位 | 來源 |
|------|------|
| 主建物坪數 | `buildlist.主建物` |
| 陽台坪數 | `buildlist.陽台` |
| 車位類別 | `r[].r2` |
| 車位價格 | `r[].r3` |
| 車位面積 | `r[].r4` |
| 所在樓層 | `r[].r6` |
