# 談價（price-negotiation）

站在**買方**立場，把「社區實價登錄 comps」＋「即時區域行情」轉成**對買方最有利、且站得住腳**的出價策略。輸出一律用**繁體中文**。詳細方法論見 `reference/談價方法論.md`；comps 統計用 `scripts/comps_stats.py`。

## 何時使用

- 使用者給了一個**大樓**物件（社區＋樓層／坪數／開價等），問「該出多少／這價格合不合理／幫我談價／砍到多少／要不要下斡旋」。
- 即使沒明講「談價」，只要語境是在決定對某大樓物件出多少價，就主動使用本 skill。

## 範圍：僅限大樓

只處理**電梯大樓／住宅大樓**。若標的是**透天、土地、店面、套房、廠辦**或無法判定為大樓 → 明確說明本 skill 僅適用大樓、**不給出價建議**，並建議改用適當方式（這類產品成交量少、定價邏輯不同）。

## 輸入

- **必要**：目標物件資訊——社區名稱（要對得上 `data/` 的檔名）、樓層、主建物或權狀坪數、車位（有無／類別／是否需要）、**對方開價**。
- **景觀對內／對外（重要）**：目標物件的**門牌或座向／朝向**，用來判定它是「對外」（朝大馬路／公園／河、永久棟距，視野開闊）還是「對內」（朝中庭／天井／鄰棟）。對外通常貴很多，務必先定位（見步驟 3 與 `reference` §3.2）；缺這項就向使用者問座向，**不臆測**。
- **選用**：棟距、格局、屋況、是否急售/滯銷、屋齡、行政區/路段。缺漏就先向使用者確認，**不臆測**。

## 流程

### 步驟 1：辨識標的（僅大樓）
確認是電梯／住宅大樓。非大樓 → 依「範圍」說明後停止，不給價。

### 步驟 2：讀社區 comps，算實坪制坪價行情
跑統計工具（關鍵字用社區名）：
```bash
uv run python .claude/skills/price-negotiation/scripts/comps_stats.py "<社區名>" --years 2
```
它會輸出：最終 comps 的實坪制坪價中位數/分位、近 2 年行情、**各樓層帶中位數**、主建物坪數分布、社區每車均價、以及被剔除的特殊交易。
- 若 `remark/` 有該社區景觀備註，會**多輸出「對外 vs 對內」兩組中位數與「對外溢價」**（含各樓層帶兩側對照、未命中規則的門牌）；無備註則略過並提示可建立。
- 找不到該社區 CSV → 告知無資料，請使用者先用 `scraper.py` 抓或確認社區名，**不臆測行情**。
- 人工複核 comps 可比性（見 `reference` §2）。

### 步驟 3：目標物件定位
拿**與目標物件條件相稱**的 comps（相近樓層帶／坪數帶／車位類別／**同一景觀側**）定錨，依樓層、景觀對內外/棟距、車位、公設比、屋齡、格局、地段做加減項。**社區 comps 的實證因子（如各樓層帶斜率、對外溢價）優先於通用參考值**（見 `reference` §3）。

**景觀對內／對外（先做）**：讀 `remark/<社區>` 規則，用目標物件的**門牌或座向**判定它屬「對外」或「對內」，再用 `comps_stats.py` 算出的**同側** comps 定錨：
- 目標**對外** → 用對外組 comps；目標**對內** → 用對內組 comps。跨側比較時套用工具算出的「對外溢價（中位數差）」校正，且**優先看相近樓層帶的兩側中位數**（避免被樓層／棟別混淆）。
- 兩組樣本太少、目標門牌對不上規則（標「未知」）、或社區無 `remark` 檔 → 說明信心較低、退回通用景觀判斷，並請使用者補 `remark`／確認座向，**不臆測對內外**。

### 步驟 4：即時查行情佐證（每次必做）
用 `WebSearch`/`WebFetch` 查當下**區域行情**與**議價空間/議價率**，以及需要的因子幅度（車位、屋齡、景觀…），與 comps 校準。附來源、區分事實與推論、剔除特殊交易、查無就說資料有限（見 `reference` §4）。

### 步驟 5：三段式報價＋話術（見 `reference` §5、§6）
由「相稱實坪制坪價基準 × 議價空間」導出起始出價／目標成交價／心理天花板，每段**回推總價**（實坪制坪價 × 目標主建物坪數 ＋ 車位合理價）並說明推導，再給壓價話術與風險。

## 誠實性規則（必守）

- 每個價格都要能溯源：用了哪些 comps、扣了什麼因子、套了多少議價空間。
- 區分「查到的事實」與「我的推論」；即時資料附來源；剔除特殊交易要講明。
- **禁止捏造**成交、行情或評價；comps 少或行情矛盾時明說信心低，不要寫成定論。
- 起始出價要積極但**站得住腳**（能用 comps 或缺點佐證），不亂砍；同時給天花板讓買方拿捏。

## 界線（非目標）

- 只服務**買方**、只談**大樓**；不做賣方估價、不做投報試算、不適用透天/土地/店面/套房/廠辦。
- 這是公開資訊彙整與議價建議，**非投資或法律意見**——結尾提醒使用者自行查證、現場看屋、詳閱合約與謄本。
